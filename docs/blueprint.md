# **App Name**: Lumina Academic

## Core Features:

- Syllabus Intelligence Engine: Upload a PDF or scan a syllabus tool that uses generative AI to extract units, detect topics, and automatically construct a semester-long study timeline.
- Personalized Semester Planner: An interactive calendar that balances college timetables, exam dates, and user availability with an automated rebalance tool for missed sessions.
- Multilingual AI Tutor: Chat-based interface allowing users to upload study materials and receive step-by-step explanations or voice-assisted coaching in English and Hindi.
- Adaptive Quiz Forge: AI-powered tool that generates practice MCQs, coding problems, and flashcards directly from the user's uploaded lecture notes and textbooks.
- Centralized Knowledge Vault: A cloud-synced database to securely store notes, scanned documents, and study plans across multiple devices using Firestore.
- Cognitive Progress Analytics: Visual dashboard tracking study hours, exam readiness scores, and highlighting forgotten topics through spaced-repetition logic.
- Academic Mastery Streaks: Gamification system with daily XP, study streaks, and performance badges to drive consistent student engagement.

## Style Guidelines:

- Primary Color: Deep Radiant Indigo (#845ef7), chosen for its association with deep focus and cognitive clarity.
- Background Color: Obsidian Shade (#0d0c11), a very dark and desaturated version of the indigo to minimize eye strain and enhance the modern high-end feel.
- Accent Color: Azure Brilliance (#85a2ff), an analogous blue hue providing vibrant contrast for critical action buttons and achievement highlights.
- Headline font: 'Space Grotesk' (sans-serif) for its tech-forward, computerized feel. Body font: 'Inter' (sans-serif) for machined, objective clarity across dense study material.
- Thinline, minimalist monolinear icons with subtle glowing effects that mirror the indigo-blue color palette.
- High-contrast modular grid system with generous negative space and semi-transparent frosted-glass containers.
- Micro-interactions using spring physics for UI transitions, creating a premium and tactile feedback experience during navigation.
